# Security Policy

## Supported Versions

Security fixes are provided for the latest tagged release and the current `main` branch.

## Reporting A Vulnerability

Please do not open a public issue for a suspected vulnerability.

Report security concerns privately through GitHub Security Advisories for this repository, or email the maintainer if advisories are unavailable. Include:

- Affected version or commit.
- Steps to reproduce.
- Expected impact.
- Any suggested mitigation.

The maintainer will acknowledge valid reports, investigate, and coordinate a fix and disclosure timeline.
