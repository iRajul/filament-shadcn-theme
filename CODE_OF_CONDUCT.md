# Code Of Conduct

This project is intended to be a welcoming, practical place to collaborate.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially when discussing bugs or design tradeoffs.
- Keep issues and pull requests focused on the project.
- Accept maintainer decisions about scope and direction.

## Unacceptable Behavior

- Harassment, abuse, threats, or discriminatory language.
- Personal attacks or repeated hostile comments.
- Publishing private information without permission.
- Spam or intentionally disruptive behavior.

Maintainers may remove comments, close issues, block users, or take other reasonable action to protect the project.
